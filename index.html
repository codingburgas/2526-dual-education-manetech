Help & Documentation

Guide

Getting Started

FileSeek is a powerful document search tool that helps you find text within your files quickly and efficiently.

How to Search

  1. Enter the directory path where your files are located
  2. Type the word or phrase you want to find
  3. Click "Search Now" to start searching
  4. View results with file paths and line numbers

Advanced Features

  • Case Sensitive: Match exact letter casing
  • Whole Word Only: Find exact word matches
  • Export Results: Download search results as JSON
  • Filter Results: Quickly filter search results
  • Dark Mode: Toggle theme for comfortable viewing

Tips & Tricks

  • Use relative paths like ./documents or ./test-data
  • Search is case-insensitive by default
  • Results show exact line numbers for easy access
  • Your recent searches are automatically saved
  • Click on recent searches to quickly re-run them

Keyboard Shortcuts

  • Enter: Submit search form
  • Ctrl/Cmd + K: Focus search input
  • Esc: Clear search
